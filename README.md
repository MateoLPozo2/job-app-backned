# Job Application Backend (MVP)

A backend for a job application platform featuring:
- JWT Authentication
- OAuth (Google)
- File uploads (resume)
- Mongoose populate
- HTTPS-ready
- CORS
- Modular MVC structure

## Getting Started
1. Clone and install dependencies
2. Set up `.env` file
3. Run with `nodemon app.js` or `node app.js`

## Tech Stack
- Node.js / Express
- MongoDB / Mongoose
- Passport.js
- Multer
